# Carry Haven 🧳

Modern e-commerce web application for browsing and exploring travel suitcases.

## 🚀 Features
- Custom SPA routing (without React)
- Dynamic page loading (fetch API)
- Responsive design
- Product catalog and UI components
- Modular SCSS structure

## 🛠️ Tech Stack
- JavaScript (Vanilla)
- Vite
- SASS (SCSS)
- HTML5
