# Carry Haven 🧳
Modern e-commerce web application for browsing and exploring travel suitcases.

## 🌐 Live Demo
(Carry Haven Demo)[https://agnieszka335.github.io/Carry-Haven/]

## 🚀 Features
- Built a **Single Page Application** from scratch (without React or other frameworks)
- Implemented custom **routing system** for seamless navigation
- Developed **dynamic data rendering** using Fetch API
- Designed and structured a scalable **product catalog system**
- Added interactive product **filtering & sorting** (e.g. price: low → high)
- **Modular SCSS structure** and **responsive design**

## 🛠️ Tech Stack
- JavaScript (Vanilla)
- Vite
- SASS (SCSS)
- HTML5

## 💡 What makes this project stand out
Unlike typical tutorial-based projects, this application focuses on core frontend fundamentals:

- Manual state handling without libraries
- Custom-built routing logic
- Component-based structure using pure JavaScript
- Real e-commerce use cases (product listing, filtering, UI interactions)

This demonstrates the ability to understand how frameworks like React work under the hood, rather than just using them.

## 📌 Project Goal
The goal of this project was to build a fully functional, modern frontend application without relying on frameworks, while focusing on:

- clean, maintainable code
- performance and user experience
- real-world e-commerce features

2. How to manage UI state without frameworks
